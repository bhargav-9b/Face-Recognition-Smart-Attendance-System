# Face-Recognition-Smart-Attendance-System
Face Recognition Smart Attendance System developed with Python and OpenCV for real-time facial identification, attendance tracking, and database management.

A Python-based attendance management system that uses face recognition technology to automatically identify users and record attendance in real time.

## Features

* Face Detection & Recognition
* Automated Attendance Marking
* Attendance Record Management
* Database Integration

## Technologies

Python, OpenCV, Face Recognition, SQLite/MySQL
